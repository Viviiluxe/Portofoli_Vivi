Tentang Saya

Profil Singkat

Halo! Saya Vivi Amanda, seorang mahasiswa Program Studi Pendidikan Teknik Informatika yang memiliki ketertarikan besar terhadap dunia pengembangan web, desain antarmuka, dan teknologi digital. Saya senang mempelajari hal baru dan mencoba mengubah ide menjadi proyek nyata berbasis teknologi.

Saya percaya bahwa kualitas karya tercipta dari proses belajar yang konsisten, sebuah prinsip yang saya pegang dalam setiap tugas, eksperimen coding, maupun proyek yang saya kerjakan.

Saat ini saya aktif mengembangkan kemampuan di bidang Frontend Development dan mempelajari dasar Backend menggunakan Node.js dan PHP. Saya bercita-cita untuk menjadi seorang Full-Stack Web Developer di masa depan.
